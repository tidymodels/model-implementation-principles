Guiding Principles for tidymodels Packages

Chapter 2 General Conventions

  • Code should follow the tidyverse style conventions

  • All results must be reproducible from run-to-run.

  • If there is a strong need for producing output during execution, there should be a verbose option that defaults to no output. {notes}
